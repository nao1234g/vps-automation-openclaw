# VPS Automation & OpenClaw Project

## Project Context
This project focuses on:
- VPS server management (ConoHa)
- Docker containerization
- OpenClaw/MultiBot deployment
- N8N workflow automation
- System administration and DevOps

## Priority Skills
- Docker best practices
- SSH security and management
- System debugging and troubleshooting
- Infrastructure automation
- Security hardening

## Model Selection
- Use Claude Sonnet 4.5 for complex system design
- Use Haiku for quick config edits and simple tasks

## Coding Standards
- Clear documentation for all scripts
- Defensive scripting (error handling, validation)
- Idempotent operations where possible
- Security-first approach

## Testing Requirements
- Test all scripts in isolated environment first
- Verify SSH connections before destructive operations
- Backup configurations before modifications
