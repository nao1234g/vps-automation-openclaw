### 設計: Prediction Memory Pipeline

```
記事生成フロー (現在):
  ニュース入力 → deep-pattern-generate.py → Ghost記事

記事生成フロー (新設計):
  ニュース入力
    ↓
  prediction_db.json を検索 (同テーマの過去予測)
    ↓
  「過去の類似予測」を記事に注入:
    "2ヶ月前、NowpatternはXを70%で予測した。結果: 的中 (Brier Score 0.09)"

  deep-pattern-generate.py → Ghost記事 (Nowpattern固有データ含む)
```

**効果**:
- 各記事に「どこにも存在しない一次データ」が自動生成される
- Googleが最重視する「Experience」シグナル（実際の予測の証拠）を全記事に挿入
- 200記事のうち過去予測と紐づくものが差別化コンテンツになる
