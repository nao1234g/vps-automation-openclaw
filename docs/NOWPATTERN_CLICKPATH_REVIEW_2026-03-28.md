## CP-07: フッター → タクソノミーガイド

### フロー

```
nowpattern.com/[任意ページ]
  └→ [フッター: タクソノミーガイド] → /taxonomy-guide-ja/
       └→ [301リダイレクト] → /taxonomy/ ❌ （誤ったページ）
```

### 評価

**完全に壊れている経路**。

| ステップ | 状態 | 問題 |
|---------|------|------|
| フッタークリック | ✅ リンク自体は動作 | — |
| リダイレクト先 | ❌ **誤りページ** | /taxonomy-guide/ ではなく /taxonomy/ に到達 |
| ユーザー体験 | ❌ 混乱 | "ガイド"を期待したら"タクソノミー検索"ページに着地 |
| 正しい /taxonomy-guide/ | ✅ 200で動作 | ただしフッターから到達不可 |

**根本原因**: Caddyfile redirects.txt の設定ミス。
```
redir /taxonomy-guide-ja/ /taxonomy/ permanent  ← /taxonomy-guide/ が正解
```
