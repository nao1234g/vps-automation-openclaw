## 私たちが目指していること

ニュースは毎日、大量に流れてきます。
「何が起きたか」は誰でも知ることができます。でも **「なぜそうなったのか」「次に何が起きるのか」** を構造的に読み解ける場所は、ほとんどありません。

Nowpatternは、世界中のニュースの裏側に繰り返し現れる **「パターン」** を見つけ出し、名前をつけ、蓄積していくメディアです。

同じ構造は、政治でも、経済でも、スポーツでも、芸能でも繰り返されます。
EUがAppleに制裁金を課すのも、大谷翔平の10年契約も、根っこにある力学は同じ **「プラットフォーム支配」** かもしれない。

その "見えない力学" に気づけるかどうかで、ニュースの読み方は根本から変わります。

## 3つのレンズ

Nowpatternでは、すべての記事を **3つのレンズ** で整理しています。

```
┌──────────────────────────────────────────────────┐
│  ジャンル           どの世界の話か                 │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ │
│  イベントタグ       何が起きたか                   │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ │
│  NOW PATTERN        なぜそうなるか ← ここが核心    │
│  （力学タグ）       記事内の分析セクションとして表示 │
└──────────────────────────────────────────────────┘
```

**ジャンル** は、ニュースの「舞台」です。政治、経済、スポーツなど、どの世界の出来事かを示します。サイトのメインナビゲーションです。

**イベントタグ** は、「何が起きたか」の型です。規制が変わった、企業が合併した、価格が急変した — 出来事の種類を分類します。記事一覧のフィルターに使います。

**NOW PATTERN（力学タグ）** は、Nowpatternの核心です。「なぜそれが起きたのか」「なぜ止まらないのか」「次に何が起きるのか」を説明する **構造的パターン** を、記事本文の中で分析・解説します。タグではなく、記事内の専用セクションとして登場します。

```
記事ページ:
┌─────────────────────────────────────────┐
│ [テクノロジー] [司法・制裁]              │ ← ジャンル＋イベントタグ
│                                          │
│ EUがAppleに2兆円の制裁金                 │
│ ──────────────────────────────           │
│ （本文）                                 │
│                                          │
│ ┌─ NOW PATTERN ────────────────────────┐│
│ │ プラットフォーム支配 × 規制の捕獲     ││ ← 力学分析欄
│ │                                       ││
│ │ Appleはアプリストアという「場」を      ││
│ │ 支配し、長年規制を骨抜きにしてきた。  ││
│ │ EUのDMAはその構造を壊す試みだ。       ││
│ └───────────────────────────────────────┘│
└─────────────────────────────────────────┘
```

## 第3層：NOW PATTERN — 力学タグ（16 = 4×4）

世界のニュースの裏側で、ジャンルを超えて繰り返される構造的メカニズム。
記事内の **「NOW PATTERN」分析セクション** で、1本につき1〜3個を解説します。

全16個は同じ形式で成り立っています：
> **「構造 × インセンティブ → 予測可能な結果」**

---

### 支配の力学 — 誰がルールを書くのか

| # | 力学タグ | English | slug | 一言 |
|---|---------|---------|------|------|
| 1 | **プラットフォーム支配** | Platform Power | p-platform | インフラを握った者がルールを書く |
| 2 | **規制の捕獲** | Regulatory Capture | p-capture | 規制者が被規制者に取り込まれる |
| 3 | **物語の覇権** | Narrative War | p-narrative | 事実より解釈の主導権を争う |
| 4 | **権力の過伸展** | Imperial Overreach | p-overreach | 覇権が力を伸ばしすぎて内側から崩れる |

**1. プラットフォーム支配 — Platform Power**

Apple、Google、SWIFT、MLB — 「場」を持つ者が参加者の運命を決める。手数料率を1%変えるだけで産業全体が揺れる。場の持ち主は自分に有利なルールを作り、参加者はそれに従うか去るかの二択を迫られる。デジタル経済だけでなく、空港、取引所、リーグ運営、芸能事務所まで同じ構造。

**2. 規制の捕獲 — Regulatory Capture**

金融庁がメガバンクに、FAAがボーイングに、文科省が大学に — 監視するはずの側が、いつの間にか監視される側の論理で動き始める。規制者は人事・情報・予算を被規制者に依存するため、時間とともに取り込まれる。「回転ドア」（官民人事交流）がこの構造を加速する。

**3. 物語の覇権 — Narrative War**

同じ経済データでも「力強い回復」と読むか「バブルの兆候」と読むかで、世論も政策も変わる。先に物語を定義した側が議論の土俵を決め、後から反論する側は常に不利になる。SNS時代には、事実の検証が終わる前に物語が確定してしまう。

**4. 権力の過伸展 — Imperial Overreach**

ローマ帝国、大英帝国、そしてアメリカの「世界の警察」— 覇権国や独占企業は成功体験を根拠に支配範囲を広げ続けるが、ある臨界点を超えると内部から維持できなくなる。軍事・外交・経済のどれか一つに集中すべきリソースが薄く広がり、抑止力も信頼性も失う。旧タグ「抑止の崩壊」はこの構造の一症状。

### 対立の力学 — なぜ争いは止まらないのか

| # | 力学タグ | English | slug | 一言 |
|---|---------|---------|------|------|
| 5 | **対立の螺旋** | Escalation Spiral | p-escalation | 対立が自己強化して止まらない |
| 6 | **同盟の亀裂** | Alliance Strain | p-alliance-strain | 利害の乖離が内部から侵食する |
| 7 | **経路依存** | Path Dependency | p-path-dependency | 過去の選択が未来の選択肢を縛る |
| 8 | **揺り戻し** | Backlash Pendulum | p-backlash | 行き過ぎた政策が反動を生み跳ね返る |

**5. 対立の螺旋 — Escalation Spiral**

報復が報復を呼び、双方が引けなくなる。関税合戦、軍拡競争、SNSの炎上 — 一度回り始めた螺旋を止めるのは、始めるより何倍も難しい。どちらの側も「相手が先にやった」と主張し、エスカレーションは合理的に見える。内部矛盾を外敵に転嫁する「スケープゴート」もこの螺旋を加速させる。

**6. 同盟の亀裂 — Alliance Strain**

NATO、OPEC、芸能事務所のグループ — 外からは一枚岩に見えても、中では利害がずれ続けている。「抜ける」という脅しだけで交渉力を持つメンバーが出現し、亀裂は危機の瞬間に一気に表面化する。同盟が長期化するほど、結成時の利害と現在の利害の乖離が広がる。旧タグ「出口のゲーム」はこの構造の表出形態。

**7. 経路依存 — Path Dependency**

過去の選択が未来を縛る。QWERTY配列、ガソリン車のインフラ、一人っ子政策 — 一度定着した構造は、たとえ非効率でも変えるコストが大きすぎて維持される。「なぜこんな不合理が続くのか？」という問いの答えは、ほぼ常に「過去にそう決めたから」。天然資源の保有が呪いにもなる「資源の呪い」もこの構造の一形態。

**8. 揺り戻し — Backlash Pendulum**

規制が厳しすぎれば緩和を求める声が噴出し、自由が行き過ぎれば締め付けが来る。制裁が発動者自身に跳ね返る「ブローバック」もこの振り子の一形態。政策・文化・市場のどの領域でも、一方向への動きは必ず反動を生む。重要なのは「振り子がいつ反転するか」を読むこと。

### 崩壊の力学 — なぜシステムは壊れるのか

| # | 力学タグ | English | slug | 一言 |
|---|---------|---------|------|------|
| 9 | **制度の劣化** | Institutional Decay | p-institutional-rot | 信頼喪失が機能不全を加速する |
| 10 | **協調の失敗** | Coordination Failure | p-collective-failure | 動くべきだとわかっていても動けない |
| 11 | **モラルハザード** | Moral Hazard | p-moral-hazard | リスクを他者に転嫁できるから暴走する |
| 12 | **伝染の連鎖** | Contagion Cascade | p-contagion | 一国の危機が連鎖的に他国へ波及する |

**9. 制度の劣化 — Institutional Decay**

裁判所、中央銀行、国際機関、芸能事務所 — 制度はハードウェアではなく「信頼」で動いている。信頼が消えると、ルールがあっても誰も従わなくなる。劣化は徐々に進行するため注目されにくく、危機が起きて初めて「実はとっくに壊れていた」と気づく。旧タグ「静かな危機」「システムの脆弱性」はこの劣化の一段階。

**10. 協調の失敗 — Coordination Failure**

気候変動、年金改革、抗生物質耐性 — 全員が「やるべきだ」と知っている。でも最初に動く者がコストを負い、動かない者がフリーライドする構造だから、誰も動かない。個人の合理性と集団の合理性が噛み合わない。「みんなで動けば全員が得をする」のに、「自分だけ動くと自分だけ損をする」。

**11. モラルハザード — Moral Hazard**

「潰れても政府が救ってくれる」銀行、「失敗しても株主が負担する」経営者、「保険があるから無茶できる」ドライバー — 自分がリスクの痛みを負わない構造が、無謀な行動を生む。2008年の金融危機はモラルハザードの教科書的事例。救済するたびに次の暴走の種が蒔かれる。

**12. 伝染の連鎖 — Contagion Cascade**

タイバーツ暴落 → 韓国通貨危機 → ロシアデフォルト。リーマン破綻 → 世界金融危機。SVB破綻 → クレディ・スイス。一つの破綻がドミノ倒しのように連鎖する。グローバル化で結びつきが深まるほど、一箇所の衝撃が全体に波及するスピードは速くなる。旧タグ「資本逃避」「金融強制」はこの伝染経路の一形態。

### 転換の力学 — 何が世界を変えるのか

| # | 力学タグ | English | slug | 一言 |
|---|---------|---------|------|------|
| 13 | **危機便乗** | Shock Doctrine | p-shock-doctrine | 非常時にしかできない改革を通す |
| 14 | **後発逆転** | Tech Leapfrog | p-tech-leapfrog | 後発が既存インフラを飛び越える |
| 15 | **勝者総取り** | Winner Takes All | p-winner-takes-all | ネットワーク効果で1位と2位の差が指数的に開く |
| 16 | **正統性の空白** | Legitimacy Void | p-legitimacy-void | 権威が正当性を失い誰もルールを信じない |

**13. 危機便乗 — Shock Doctrine**

パンデミックでリモートワーク定着、9.11でセキュリティ法制強化、震災で原発見直し — 「平時なら絶対に通らない」政策や変化が、危機の混乱の中で一気に実現する。危機は「窓」を開くが、その窓を誰が利用するかで結果は大きく変わる。準備していた者だけが、窓が開いた瞬間に動ける。

**14. 後発逆転 — Tech Leapfrog**

アフリカのモバイル決済、中国のEV、インドのデジタル公共基盤 — 「遅れている」ことが逆に有利になる。古いインフラの維持コストがない分、最新技術を白紙から導入できる。先進国は過去の成功体験（経路依存）に縛られ、後発国は一気に追い越す。

**15. 勝者総取り — Winner Takes All**

Google検索、Instagram、英語、ドル、プレミアリーグ — ネットワーク効果により、1位と2位の差は足し算ではなく掛け算で開く。ユーザーが多いからさらにユーザーが集まり、2位は永遠に追いつけない。テック、通貨、言語、スポーツリーグ、SNSプラットフォーム — あらゆる分野で「1位以外に価値がない」構造が拡大している。

**16. 正統性の空白 — Legitimacy Void**

国連が停戦を決議しても誰も従わない。中央銀行が「インフレは一時的」と言っても市場が信じない。既存の権威が正当性を失ったとき、そこに「空白」が生まれる。その空白を埋めるのは、ポピュリスト、テック企業、暗号資産、あるいは力ずくの実効支配。アラブの春、ビットコインの台頭、トランプ現象 — すべて「旧秩序の正統性が空っぽになった瞬間」に起きている。

## 記事のタグ付け例

### 例1：「EUがAppleに2兆円の制裁金」

```
ジャンル:      テクノロジー / 経済・金融
イベントタグ:  司法・制裁 / 標準化・独占
```

> **NOW PATTERN: プラットフォーム支配 × 規制の捕獲**
>
> Appleはアプリストアという「場」を支配し（プラットフォーム支配）、
> 長年規制を骨抜きにしてきた（規制の捕獲）。
> EUのDMAはその構造を壊そうとする試み。

### 例2：「大谷翔平、ドジャース10年7億ドル契約」

```
ジャンル:      スポーツ / ビジネス・企業
イベントタグ:  資本移動・投資 / 競争・シェア争い
```

> **NOW PATTERN: プラットフォーム支配 × 勝者総取り**
>
> MLBという「場」が放映権収入を独占し、一部の球団に資金が集中する構造
>（プラットフォーム支配）。トップ選手に資金が集中し、2番手との差が
> 加速度的に開く（勝者総取り）。

### 例3：「日本の出生率が過去最低を更新」

```
ジャンル:      社会・人口
イベントタグ:  人口動態・ライフスタイル
```

> **NOW PATTERN: 協調の失敗 × 経路依存**
>
> 政府・企業・個人の三者が「誰かが先に動くべき」と待ち続けている
>（協調の失敗）。1980年代の都市集中政策、終身雇用モデル、
> 核家族化の定着が30年後の今の出生率を規定しており、
> 政策の方向転換が構造的に困難になっている（経路依存）。

### 例4：「中国の人口減少、世界経済に波及」

```
ジャンル:      経済・金融 / 社会・人口
イベントタグ:  市場変動 / 人口動態・ライフスタイル
```

> **NOW PATTERN: 経路依存 × 勝者総取り**
>
> 一人っ子政策（1979-2015）という過去の選択が、40年後の
> 労働人口を決定的に縛っている（経路依存）。労働人口を
> 維持できる国（インド、米国）に資本と企業が集中し、
> 人口減少国との差が加速度的に開く（勝者総取り）。

### 例5：「ドイツ、移民受け入れ大転換」

```
ジャンル:      政治・政策 / 社会・人口
イベントタグ:  規制変更 / 人口動態・ライフスタイル
```

> **NOW PATTERN: 危機便乗 × 対立の螺旋**
>
> 労働力不足という「静かな危機」を利用して、平時なら通らない
> 移民法改正を一気に実現した（危機便乗）。一方で移民推進派vs
> 反対派の対立が政策の振り子を大きく振らせ、次の政権交代で
> 揺り戻しが来る可能性がある（対立の螺旋）。

### 例6：「韓国の兵役制度見直し、少子化で限界」

```
ジャンル:      地政学・安全保障 / 社会・人口
イベントタグ:  公共政策・税制 / 人口動態・ライフスタイル
```

> **NOW PATTERN: 経路依存 × 協調の失敗**
>
> 徴兵制は朝鮮戦争時代の選択の延長（経路依存）。
> 「安全保障のために維持すべき」と「経済のために縮小すべき」の
> 間で、どの政権も本格改革を避けて次の政権に先送りしている
>（協調の失敗）。

### 旧イベントタグ → 新イベントタグへの統合先

| 旧（30個） | 新（22個） | 統合の理由 |
|-----------|-----------|-----------|
| 規制強化 + 規制緩和 | **規制変更** | 方向は違うが記事の型は同じ |
| 司法・訴訟・判決 + 監督・制裁・処分 | **司法・制裁** | どちらも「権力による裁定」 |
| 選挙・政権・政治合意 | **選挙・政権** | 短縮のみ |
| 国際関係・地政学・制裁 + 安全保障・危機管理 | **地政学・安全保障** | 国際問題は安全保障と不可分 |
| 景気循環・需要変動 + 価格変動・インフレ/デフレ | **市場変動** | どちらも「市場の動き」 |
| 企業業績・収益構造 + コスト構造変化・効率化 | **収益・コスト変動** | 収益とコストは表裏一体 |
| M&A・提携・統合 + 参入/撤退・事業再編 | **事業再編・M&A** | 企業の構造変化 |
| 技術ブレイクスルー + 普及・コモディティ化 | **技術進展** | 「登場→普及」は技術の一生 |
| 炎上・価値観衝突 + 流行・ナラティブ転換 | **世論・価値観変動** | 世論の動き |
| 標準化・プラットフォーム支配 | **標準化・独占** | 力学タグとの命名衝突を解消 |
