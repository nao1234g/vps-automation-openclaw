## 1. note アルゴリズム完全理解（2025年最新）

### 仕組み（3要素）
```
表示順位 = 関連性 × エンゲージメント × アカウント信頼度
```

### 最重要：初速の法則
- **投稿後1〜2時間が全てを決める**（この時間のスキ・コメント・シェアで9割決まる）
- 初速が出ると「おすすめ」に表示 → 連鎖的に読まれる
- **初速を作る方法 = X投稿と同時に連携ツイートを打つ**

### noteアルゴリズムが評価するもの（優先順）
1. **スキ率**（読んだ人のうち何%がスキしたか）
2. **コメント数**（返信することで「活動的」と判定）
3. **シェア数**（X連携シェアが最も効果的）
4. **読了率**（途中離脱が少いほど評価UP）
5. **フォロワーの反応速度**（フォロワーが早く反応するほど良）
6. **投稿頻度**（継続的に投稿するアカウントが優遇）

### 投稿タイミング（データ基準）
- **最適時間帯**: 朝7〜9時、昼12〜13時、夜20〜22時（日本時間）
- **最も読まれる曜日**: 月曜・火曜・木曜
- **避けるべき**: 深夜0〜5時、土曜夜（競合多い）

### タグ戦略
- **推奨数**: 5〜10個（公式推奨5個だが実態は多め）
- **組み合わせ**: 人気タグ（検索ボリューム大）+ ニッチタグ（競合少）
- **AISA推奨タグ**: `#地政学` `#AI` `#経済` `#国際情勢` `#暗号資産` `#ニュース解説` `#アジア`

## 2. 伸びる記事の条件（note公式データ + 調査）

### タイトルのルール（絶対守る）

| 項目 | ルール | 理由 |
|------|--------|------|
| **文字数** | **25〜30文字** | SNSで省略されない限界 |
| **数字を入れる** | 必須（可能なら） | 具体性でCTRが上がる |
| **答えを先に言う** | 推奨 | 読者が「読む価値」を瞬時に判断 |

### 最強タイトルフォーマット（AISAバージョン）

```
❶ 数字型:    「香港が36社から3社だけを選ぶ理由」
❷ 逆説型:    「AIが経済を救うのではなく、壊す構造」
❸ 問い型:    「なぜ日銀は今、金利を上げられないのか」
❹ ニュース型: 「米中関税交渉、水面下で起きていること」
❺ 予測型:    「2026年、アジア暗号資産規制の次の一手」
❻ 解説型:    「ステーブルコインが通貨になる日の条件」
```

### 記事構成テンプレート（AISA標準）

```
【フック段落】（200〜300字）
  → 「今〇〇が起きている。ほとんどの人が知らない理由がある。」
  → 衝撃的な数字1つ＋なぜこれが重要かを1行

【事実・データ】（箇条書き3点）
  → 日付・数字・固有名詞を必ず入れる
  → グラフや図解があれば挿入

【AISAの視点】（記事の核心）
  → 「表面上は〇〇に見えるが、本質は〇〇だ」
  → 他のメディアが言っていないことを1つ必ず入れる
  → 歴史・構造から読む（AISAのブランド）

【インサイト・予測】
  → 「これが意味すること」「6ヶ月以内に起きること」

【まとめ + CTA】
  → 有料マガジンへの誘導（メンバーシップ読者へのメリット明示）
```

### 文字数のガイドライン
- **無料記事**: 2,000〜4,000字（価値を伝えて満足させる）
- **有料記事**: 4,000〜6,000字（深層分析、予測、独自データ）
- **注意**: 文字数と売上の相関はほぼゼロ（価値提供が全て）

### AI記事の扱い
- **AIが書いた記事をそのままは出さない**（noteが意図的にアルゴリズムで排除）
- 必ずAISAの独自視点・判断・経験を入れる
- 「AIで量産したコンテンツ」に見えない書き方が2026年の生存条件

## 3. X（Twitter）戦略ルール

### 基本フォーマット：7ツイートスレッド

```
Tweet 1【フック】    数字 or Bold statement（最初3秒で掴む）
Tweet 2【データ1】   画像（チャート/地図）＋ 事実
Tweet 3【データ2】   事実・タイムライン
Tweet 4【視点1】    「本質はこれだ」
Tweet 5【視点2】    「他が見落としていること」
Tweet 6【インサイト】「6ヶ月以内に起きること」
Tweet 7【CTA】      「深掘り分析はこちら → [note URL]」
                    #地政学 #AI #経済 #AISA
```

### Xアルゴリズム（2025年）
- 投稿後**最初5〜10分**の返信数が最も重要
- 返信 > ブックマーク > RT > いいね（効果の順）
- **画像付きは2倍のリーチ**（必ず画像を入れる）
- X Premiumで追加リーチブースト（加入済み✅）
- リンクは本文内よりリプライに入れた方がリーチ向上（ただしPremium加入後は本文でも可）

### 投稿パターン（週間スケジュール）
- **月・水・金**: 7ツイートスレッド（主要分析）
- **火・木・土**: 単体ツイート（短いインサイト、データ紹介）
- **日**: まとめ or 週次振り返り
- **頻度**: 1日1〜2投稿が最適（過多は逆効果）

### 2パターンを交互に
- **Aパターン（価値提供型）**: スレッドで完結。noteへの誘導なし
- **Bパターン（誘導型）**: Tweet 5〜6で「続きはnoteで →」

## 6. 絶対にやってはいけないこと（ドリフト防止ルール）

### ❌ NGリスト
- AI生成記事をそのままnoteに投稿（アルゴリズム排除）
- 画像なしでXにスレッドを投稿（リーチ半減）
- タイトルを30文字以上にする（SNSで省略される）
- 毎日投稿するが品質を下げる（週3〜4本の高品質 > 毎日低品質）
- 投稿後にX連携を忘れる（初速が作れない）
- AISAの独自視点なしに事実だけ並べる（Morning Brewに学ぶ：視点が命）
- 「crypto regulation」だけに絞る（ブランドが古い）
