## PVQE — 成果の公式（完全版）

**Outcome = P（判断精度） × V（改善速度） × Q（行動量） × E（波及力）**

掛け算。どれか一つがゼロなら全体がゼロ。

### P（判断精度）— 正しい方向を選ぶ力

Pがゼロになる条件（いずれかに該当したらPはゼロ）:
- 推測・希望的観測でコードを書く / 報告する
- 確認なしに実装する（「こういう理解でいいですか？」を省く）
- 廃止済み概念（@aisaintel / AISA）を参照する
- UIレイアウトを承認なしに変更する
- ミスを後から報告する（ノーサプライズ違反）

Pが最大になる条件:
- 実装前にWebSearch + 公式ドキュメントで裏付け
- 事実の分類（Unshakeable / Surface / Assumed / Reported / Wishful）を使って判断
- ミスは即記録 → KNOWN_MISTAKES.md → auto-codifier.py で物理ブロック化

### V（改善速度）— ループを速く回す力

- エラー → 記録 → パターン化 → ブロック のサイクルを24時間以内に完了
- AGENT_WISDOM.md は毎週日曜自動更新（evolution_loop.py）
- 同じミスを2回繰り返した時点で V=0

### Q（行動量）— 投入リソース

- 1日200記事（JP100 + EN100）: コンテンツ量の最低基準
- NEO-ONE/TWO: 24時間稼働
- Hey Loop: 1日4回の情報収集
- 0件アラート: 記事ゼロは30分以内に検知

### E（波及力）— 世界への伝達倍率

- X(@nowpattern): 100投稿/日、X Premium加入済み
- note + Substack: 補助配信
- /en/predictions/ : 英語圏への信号
- hreflang + /en/ URL構造: Googleへの言語シグナル
