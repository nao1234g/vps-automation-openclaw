【重要タスク】Nowpattern タクソノミー v3.0 — 全35記事の整合性レビュー＆修正

## 背景
力学タグが20→16に統一されました。既存35記事のタグと本文の整合性を取る必要があります。

## 指示ファイル
/opt/shared/article_review_v3_instructions.json を読んでください。各記事の問題点と修正ガイドが書かれています。

## レビュースクリプト
/opt/shared/scripts/article_review_v3.py --report でレポート生成済み。

## やること（優先順）

### 1. 自動修正可能なタグ（7箇所）
以下の旧タグを持つ記事のタグをGhost APIで新タグに変更：
- Institutional Rot → Institutional Decay
- Blowback → Backlash Pendulum

### 2. レビュー必要なタグ（旧力学タグ）
以下の旧タグは記事の内容を読んで最適な新タグを選ぶ：
- Exit Game → Winner Takes All / Path Dependency / Escalation Spiral のどれか
- Resource Power → Platform Power / Path Dependency のどれか
- Financial Coercion → Path Dependency / Regulatory Capture のどれか
- Deterrence Failure → Escalation Spiral に吸収（すでにEscalation Spiralがある場合は Imperial Overreach 等を検討）
- Capital Flight → Contagion Cascade
- Systemic Fragility → Contagion Cascade or Institutional Decay

### 3. 日本語独自タグ（8記事 — 最初に投稿された日本語記事）
これらはタクソノミー外のカスタムタグを使っている。v3.0のジャンル(12)+イベント(19)+力学(16)から正しいタグに振り直す。

### 4. 記事本文の整合性
タグを変更した記事のNOW PATTERNセクションで旧タグ名が使われている場合は、新タグ名に書き換える。分析の論旨が新タグの定義と合っているか確認。
