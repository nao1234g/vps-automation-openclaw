# 記事テーマローテーション設定

## 📅 曜日別テーマ

| 曜日 | テーマ | ターゲット |
|------|--------|----------|
| 月 | AIツール活用 | 副業志望者 |
| 火 | プログラミング | 初心者〜中級者 |
| 水 | 投資・資産形成 | 20-40代 |
| 木 | フリーランス | 独立検討中 |
| 金 | ノーコード開発 | 非エンジニア |
| 土 | AI画像生成 | クリエイター |
| 日 | 週間まとめ | 全員 |

## 📊 N8Nワークフロー設定

ファイル: `multi-theme-workflow.json`

テーマはランダムで選択されますが、曜日別に固定したい場合は
`$now.weekday` を使って条件分岐してください。
