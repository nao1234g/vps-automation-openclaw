# Note + X 自動化 プロンプト集

## 1. キーワードリサーチ用プロンプト

```
今日のトレンドキーワードを3つ提案して。
以下の分野から選んで：
- AIツール・ChatGPT活用
- 投資・資産運用
- 副業・フリーランス

各キーワードについて：
1. キーワード名
2. なぜ今注目されているか（1文）
3. 想定読者層
```

---

## 2. Note記事生成用プロンプト

```
以下のキーワードで2000〜3000字のNote記事を書いて。

キーワード: [ここにキーワード]

要件：
- SEO最適化されたタイトル（32字以内）
- 見出し（H2, H3）を適切に使用
- 読みやすい文体（です・ます調）
- 具体例や数字を含める
- 最後にまとめと行動喚起

構成：
1. 導入（興味を引く問いかけ）
2. 本文（3-4セクション）
3. まとめ
4. 関連おすすめ（アフィリエイト導線）
```

---

## 3. X投稿用プロンプト

```
以下のテーマで有益なツイートを作成して。

テーマ: [AIツール / 投資 / 副業]

要件：
- 140字以内
- 読者の関心を引くフック
- 具体的な数字や事実
- ハッシュタグ2-3個
- 絵文字を1-2個使用

トーン: 親しみやすく、専門的すぎない
```

---

## 4. アフィリエイト記事用プロンプト

```
以下の商品/サービスのレビュー記事を書いて。

商品名: [商品名]
カテゴリ: [AIツール / 投資サービス / 副業ツール]
アフィリエイトリンク: [リンク]

構成：
1. こんな悩みありませんか？（問題提起）
2. [商品名]とは？（概要）
3. 実際に使ってみた感想（体験談風）
4. メリット・デメリット
5. こんな人におすすめ
6. 始め方・登録方法
7. まとめ（CTAボタン）

文字数: 2500〜3500字
```
